# LabooAS
Доступний додаток для Android

Інструкція із встановлення:
1) Вибрати файл LabooASMobile.apk
2) Натиснути в мені справа ... та вибрати пункт Download
3) Підтвердити завантаження файлу
4) Відкрити додаток у завантажених та надати доступ до встановлення додатків з цього джерела. Підтвердити безпечність додатку
5) Вибрати Встановити
6) Система запропонує перевірити додаток на наявність небезпек - Погодитись
7) Після встановлення вибрати Відкрити додаток
8) Надати додатку необхідні дозволи. Для цього можливо знадобиться перейти в Налаштування - Програми - Laboo AS Mobile - Дозволи програм. Додаток потребує дозволів до Геоданих та Пристрої поблизу, Доступ до файлів
